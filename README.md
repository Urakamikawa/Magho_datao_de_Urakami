# Magho_datao_de_Urakami
# 日麻竞技数据管理系统 (Mahjong Stats System)

这是一个专为雀友设计的轻量化、可视化日麻战绩统计工具。支持多玩家数据汇总、对局趋势分析以及 H2H (头对头) 胜率对比。

## 在线访问入口（de Salicis League）
** [点击此处进入数据管理后台](https://urakamikawa.github.io/Magho_datao_de_Urakami/)**

---

##  核心功能
- **自动化统计**：录入点棒后自动计算 UMA、顺位、累计点数、均顺等核心指标。
- **趋势可视化**：内置折线图，直观展示每位玩家近 10/20/40 场的顺位走势。
- **对抗分析**：一键查询任意两位~~仇家~~玩家之间的相对胜率。
- **起家统计**：详细记录东、南、西、北各位置的起家频次。
- **数据持久化**：基于 JSON 文件的导出与读取，支持手动备份与云端同步。（踹我更新）

## 使用指南

### 1. 录入对局
- 进入“对局录入管理”页面。
- 输入对局日期，并按 **[东 -> 南 -> 西 -> 北]** 的顺序选择玩家并输入结算点棒。
- **注意**：点棒输入采用“百点制”，例如 `25000` 点请输入 `250`。总和必须等于 `1000` 才能保存。

### 2. 更新云端数据 (GitHub 同步)
由于本系统为静态页面，数据更新需要手动覆盖 JSON 文件：
1. 在网页录入完成后，点击 **“💾 导出 JSON”** 下载最新的 `mahjong_data.json`。
2. 回到本 GitHub 仓库，点击 `mahjong_data.json` 文件。
3. 点击右上角的 **铅笔图标 (Edit this file)**。
4. 将下载的 JSON 文件内容全选覆盖粘贴进来，点击底部的 **“Commit changes”**。
5. 刷新在线访问页面，数据即刻同步。

## 第一章 基本规定

竞技由一桌四人进行，每东南二风一半庄为一回战。

半庄内第一圈为东场，第二圈为南场。

1.   麻将牌由数牌108枚（万·饼·索1~9各4枚）与字牌28枚（东·南·西·北·白·发·中，其中 东·南·西·北 为风牌，白·发·中 为三元牌）共计136枚为组成。
2.   一副麻将牌内设5万、5饼、5索各一枚为赤牌。在对局前可约定所设置的赤牌的枚数及构成。
     -   除5万、5饼、5索的三赤五规则外，亦有两种标准赤牌设置：
         1.   四赤五：设5万、5索各一枚，5饼设两枚。
         2.   赤三：设3万、3饼、3索各一枚，赤3索和了不记绿一色。
3.   每名竞技者在对局开始时持有万点棒1根，五千点棒2根，千点棒4根，百点棒10根，共计两万五千点。
4.   色子一共两枚，均为六面色子。
5.   起家右手处放置起家标记牌（即东南牌）以标记起家与当前场次。
6.   对局卓中央放置风盘，风盘四边有槽，作指示各家风位，放置立直供托及规整牌河之用。
7.   为告知对局者，在进行 吃、碰、杠、立直、荣和、自摸和 之竞技行为时必须发声。



## 第二章 牌、宝牌

### 手牌、王牌、海底牌

1.   原则上手牌应保持为十三枚，每进行一次杠则增加1枚。

2.   开牌后牌墙末端应保留十四张牌，称王牌。

3.   王牌前最后一张牌称海底牌。摸海底牌者不得开杠。

4.   摸海底牌者，或摸海底前一张牌后开杠者，若自己未和牌，则必须打一张牌，该牌称河底牌。其余玩家不得吃、碰、杠这张河底牌。

5.   开杠后，需从王牌末尾摸一张牌补充手牌，该牌称岭上牌。岭上牌由王牌最末端一墩上段开始，每开一杠依次取一张。岭上牌必须在明示杠子之后再取。

### 宝牌

1.   本宝牌：开牌结束后，翻开王牌末端倒数第三墩之上段牌，称宝牌指示牌，该牌的下一顺次牌为本宝牌（，三元牌以白·发·中之顺序循环）。

2.   杠宝牌：宝牌随开杠增加，第一次杠后立即翻开由王牌末尾数起第4墩上段那张牌，作为新宝牌指示牌。之后的杠重复此行为，第四次杠翻开由王牌末尾数起第7墩上段牌。抢杠发生时，杠视为不成立，无杠宝牌及里宝牌产生。

3.   里宝牌：立直家和牌翻里宝牌。所有表宝牌指示牌的下段牌为里宝牌指示牌。杠宝牌亦有里宝牌。

4.   上记所有宝牌皆记为一翻，不计为役。



## 第三章 竞技流程

1.   **座次与亲家**

     -   座次与亲家决定规则

         1.   四名对局者任意就坐。

         2.   由任一一名对局者将东、南、西、北、1、2（为任一一色一、二之索、饼、万）六张牌洗好，并横向并排放置。

         3.   由任一另一名对局者掷色子，色子指向^†^的对局者将洗好的六张牌翻开，并将1、2就近放置于六张牌的两侧，此时，应当表现为 1·四·枚·风·牌·2 或 2·四·枚·风·牌·1 之形态。

         4.   由翻牌的对局者开始，每名按1·2所指示的顺序（1<2为顺时针、2>1为逆时针）从色子值的奇偶侧（奇自1起，偶自2起）开始取走一枚牌。

         5.   由拿取东风的玩家为准，其他玩家按逆时针呈 东·南·西·北 的顺序就座。

              †：自掷色者位置起为1，逆时针序数出色子之值。

     -   以起家为始，亲家以逆时针顺序在玩家间轮换。若亲家未正确轮换，则应在发现时立刻换回正确的亲家。亲家未正确轮换的局数作废，若期间发生过点棒交接，则按交接前原样恢复。

2.   **洗牌与码牌**

     1.   洗牌（将所有牌充分打乱混合）的行为在竞技开始时进行。
     2.   对局者在洗牌结束后应在面前码起一座牌山，称为码牌。
     3.   牌山一共四座，每座为上段下段组成的两层，每层17张牌。

     4.   对局者应在开始时将自己面前的牌山推成口字形，此行为称堆牌。

     5.   若堆牌时破坏牌山，则视情况给予处罚。

3.   **配牌与开牌**

     1.   由亲家打出两枚色子，从色子指向的对局者前方的牌山中，自该对局者的右手侧数出色子值的墩数值的牌，以此为牌山始；另一侧为牌山末，为最后一墩王牌的位置。
     2.   由庄家开始，按逆时针，从牌山始，一人抓起一两墩两段合计四张牌，该过程重复三次，直至所有对局者拿到十二张牌为止。
     3.   随后，庄家在牌山始抓起一枚牌，并在次次一墩牌抓取上段牌；随后，按逆时针，一人抓取一枚牌。至此，配牌结束。
     4.   配牌结束后，由距离王牌最近的对局者翻开王牌末端倒数第三墩之上段牌作为宝牌指示牌；同时，将王牌末端第一墩牌之上段牌按牌山方向置于下段牌逆时针侧。
     5.   对局者将配牌翻开的过程即为开牌。对局者开牌前出现相公不处罚，开牌后出现相公按照犯规行为进行处罚。

4.   **取牌**

     -   取牌行为指以下5种：自摸、吃、碰、杠、荣和。

5.   **摸打**

     1.   摸打指从摸牌开始至打牌为止的动作，必须按照先摸后打的顺序进行。

     2.   摸打的一巡指从自己打牌开始，到下一次自己摸牌之前的时间。

     3.   不可将其它牌握在手中的同时进行摸打。

     4.   摸打动作必须由单手进行，原则上使用玩家的惯用手。

     5.   为不影响竞技，应尽量避免摸打速度过快或过慢。

6.   **摸牌**

     1.   摸牌指从牌山中取得一张牌的动作。
     2.   以摸牌为目的进行摸牌动作视为摸牌行为的开始，当一名玩家触及牌山时，任何玩家不可再进行吃、碰、杠、荣和行为。
     3.   摸牌动作需在上家舍牌动作结束并确认后方可进行。
     4.   在上家舍牌动作结束前摸牌，即先摸行为。
     5.   摸到的牌应置于手牌上方或手牌两侧，不可将摸到的牌插入手牌中；若确实需要插入手牌，应如实报出手切摸切。

7.   **打牌**

     1.   打牌指摸牌后将手牌中任意一张牌置于牌河中的行为，也包括吃、碰、杠的行为结束后将任意一张牌置于牌河中的行为。

     2.   打出的牌触及牌河的瞬间即视为舍牌，不可再取回。
     3.   牌河应处于自己面前的牌山之前的区域。部分情况下，将牌置于其它区域时也可判定为打牌动作。
     4.   为文明起见，请尽量避免强打（即用力打牌）。

8.   **舍牌**

     1.   舍牌应从左至右贴上风盘放置，作为牌河。

     2.   舍牌一列六张。

     3.   不可触摸他家舍牌。
     4.   在非摆正牌河的情况下，请避免触及牌河，以免认定为出千。
     5.   点棒支付结束前，禁止打乱牌河。

9.   **和牌**

     -   见第五章。

10.   **流局与听牌**

      1.   流局指未出现和牌者的局数。在任何情况下皆不允许途中流局。

      2.   听牌指有可能和牌的状态，若自己的待牌已经全部用于自己的手牌及副露牌中（即虚听，指从自己的手牌上看，所听的是第五张所待的牌），则不承认为听牌。

      3.   流局的听牌者应打开自己的手牌并进行听牌宣言，无论手役有无（即允许形形式听牌）。

      4.   听牌宣言原则上应按照东、南、西、北家顺序依次进行。

      5.   听牌罚符总计应为3000点，由未听牌者向听牌者支付。
      6.   听牌者可自行决定是否进行听牌宣言。裸单骑亦可宣告未听，但当立直者宣布未听，按诈立处理。
      7.   流局时立直供托不返还，置于风盘中央，计入下一局。

11.   **连庄与轮庄**

      1.   连庄指亲家未发生变动，继续下一局。

      2.   轮庄指亲家轮换至下家。

      3.   亲家和牌或听牌时连庄，除此之外全部轮庄。

12.   **本场**

      1.   连庄或平局时，下一局视作加一本场，之后类此叠加。对局重开不计入本场。

      2.   本场数由由亲家将百点棒置于其右手侧桌缘表示。

      3.   每一本场为和牌点数加算300点。

      4.   子家和牌时本场数清零。

13.   **竞技结束**

      1.   和牌、流局、有对局者受对局重开的处罚时为终局，除此之外无途中流局。

      2.   即使有玩家在半庄途中失去所有点棒（被击飞），竞技仍继续进行到最终局为止。

      3.   半庄结束以最后一位庄家下庄为标准。

           

## 第四章 竞技细则

1.   **竞技行为**

     1.   需要发声的竞技行为以发声视作开始，以动作结束视为终止，发声与动作可以同时开始，但禁止先行动作后发声。若仅有发声而无动作，视为空行为；若仅有动作而无发声，视为露牌。

     2.   发声为：

          |   行为   | 自摸 | 荣和 |  吃  |  碰  |  杠  |
          | :------: | :--: | :--: | :--: | :--: | :--: |
          | 日语发声 | ツモ | ロン | チー | ポン | カン |
          | 汉语发声 | zìmō |  hú  | chī  | pèng | gàng |

     3.   同桌对局者应当协助确认他家的竞技行为，彼此持有保证对局正确规范进行之义务。

2.   **行为顺序**

     1.   竞技行为的优先顺序如下：
          1.   荣和
          2.   碰、杠
          3.   吃
     2.   若碰、杠、吃同时发生，则按优先顺序进行处理。
     3.   同时发生是指发声后，取牌前的时间段内。
     4.   如有需要，可向其他对局者示意，以对是否对某舍牌行竞技行为做长考。
     5.   当未向其他对局者示意时，在吃的行为开始之后进行的碰、杠、荣和视为不成立。

3.   **吃**

     1.   吃指发声之后（或同时），展示手中的搭子，随后取上家的舍牌做成顺子，根据副露规范放置于自己的右侧，并打出任意一张牌的行为。
     2.   若吃、碰同时发声，则吃不成立。该未成立的吃不视作空行为。
     3.   若在他家碰、杠取牌后发声吃，视为空行为。

4.   **碰**

     1.   碰指对他家的舍牌后进行发声之后（或同时），展示手中的对子并取得该牌做成刻子，根据副露规范放置于自己的右侧，并打出任意一张牌的行为。
     2.   若吃、碰同时发声，碰成立。
     3.   若碰在他家吃取牌后发声，视为空行为。

5.   **杠**

     -   杠分为明杠及暗杠两种。

     1.   暗杠

          1.   暗杠指摸牌后进行发声，展示手牌中（包括摸到的牌）四张同种牌，按一定规范放置于自己的右侧，并在王牌末尾取一张岭上牌的行为。

          2.   方式规范有两种，依对局者喜好决定：

               -   将中间两张牌朝下，另两张牌朝上并排放置。

               -   将中间两张牌朝上，另两张牌朝下并排放置。

          3.   立直后可以进行暗杠，须满足以下条件，否则记为违规开杠：

               1.   不可对立直时已存在的手牌进行暗杠（送杠）。
               2.   不得改变已有手牌的组成（分解）方式及听牌分解方式。
               3.   不得改变听牌数目。

          -   立直后违规开杠实例：

                1.   对已存在的手牌进行送杠：111123摸4杠1。
                2.   -   改变已有手牌的组成方式：444555666摸4、5、6任意一张开杠：丢失了一色三通顺的分解方式。
                     -   改变听牌分解方式：5557888摸5开杠：仅保留了555 78 88的分解方式，丢失了55 57 888的分解方式。

                3.   改变听牌数目：5557888摸8开杠：从6 7 9三面听变为6 7两面听。

     2.   明杠

          1.   明杠分为加杠于大明杠两种。

               -   加杠指摸牌后，从手中（包括摸到的牌）拿出一张同种牌加于自己的明刻子之上，并在王牌末尾取一张岭上牌的行为。

               -   大明杠指对他家的舍牌进行发声之后（或同时），展示自己手中的与该舍牌相同的暗刻子并取得该牌做成杠子，根据副露规范放置于自己的右侧，并在王牌末尾取一张岭上牌的行为。

          2.   若吃、杠同时发声，杠成立。

          3.   若杠在他家吃取牌后发声，视为空行为。

     3.   一局中总计最多开四次杠。

     4.   杠宝牌在杠子明示并进行确认后即翻开，离王牌最近的对局者应尽快翻开杠宝牌指示牌。若杠因抢杠而未成立，则不翻开杠宝牌指示牌。

     5.   除非是十三幺九或臭满，否则对暗杠的抢杠都不成立。

6.   **副露**

     1.   副露牌指吃、碰、杠后置于牌桌右侧的公开牌。

     2.   副露规范如下：

          1.   明顺子：

               吃到的牌横置，并放置于由手牌公示出来的搭子的左侧。

          2.   明刻子：

               碰到的牌横置，并放置于由手牌公示出来的对子中。

               若碰上家的牌，则放置于对子左侧；若碰对家的牌，则放置于对子中间；若碰下家的牌，则放置于对子右侧。

          3.   明杠子（大明杠）：

               杠到的牌横置，并放置于由手牌公示出来的暗刻中。

               若杠上家的牌，则放置于刻子左侧；若杠对家的牌，则放置于中间任一一张，推荐放置于左数第二张；若杠下家的牌，则放置于刻子右侧。

          4.   加杠子：

               加杠牌置于杠源指示牌上方。

          5.   上记四条，若进行多次副露，则将副露牌放置于自己的右侧，按副露的先后顺序由近至远纵向放置。暗杠子亦适用于本条。

     3.   吃、碰、杠的指示牌与指示方向未正确放置时，若该局同桌对局者发现，应指示改正。

7.   **立直**

     -   立直包括立直、两立直、开立直及开两立直。

     1.   立直仅可在门清状态下进行，只要立直宣言牌未被荣和即成立。

     2.   立直宣言需在发声之后（或同时），并将打出牌横置于牌河中。随后取出一根千点棒作为供托棒（立直棒）放置于风盘供托槽中。

     3.   立直可在任一巡目进行，除非当前无余牌。

          -   当且仅当第一巡无任何鸣牌（包括错行为）时，才能进行两立直及开两立直。

     4.   开立直及开两立直应展示手牌中所有与听牌相关的手牌，当展示手牌不完全，视为诈立，当展示手牌超出范围——该范围应包含所有可能的听牌分解方式——记为露牌。

     5.   若立直宣言牌被吃、碰、杠，则下一巡的打出牌需横置。

     6.   供托：

          1.   供托指立直时所交付于风盘的千点棒，也包括犯规受罚取供托惩罚所交付的千点棒。
          2.   供托由首位和牌者取得。
          3.   若本局流局，则供托归下一局，由下一局的首位和牌者取得。
          4.   若南四局流局，则立直棒归于当前一位所有。
          5.   若南四局流局时存在同分并列一位，供托均分。
          6.   若供托点数不可整除，则离起家较近者取余数，其余部分均分（每根千点棒为400、300、300；每三千点时，点棒均分一次）。

     7.   立直一经发声即不可撤回，若发声后无立直行为，视为空行为。

     8.   立直后允许见逃和牌，但之后视为立直振听。

     9.   立直后允许暗杠。

          

## 第五章 和牌

-   和牌时必须有一翻以上的役。所有情况下皆保持一翻缚。

1.   **和牌**

     1.   和牌牌型有：
          -   四面子带一雀头
              -   面子为顺子（三张连续数牌）或刻子（三张相同牌或已开杠的四张相同牌）
              -   雀头为一对相同牌
          -   七对子
          -   龙七对子
          -   十三幺九
          -   臭满
     2.   和牌的手段有以下两种：
          -   自摸和：通过自己的摸牌和牌（包括岭上自摸）。
          -   荣和：通过他家的舍牌（包括暗杠与加杠牌）和牌，舍牌者视为放铳。
     3.   一局仅允许一人和牌。
          1.   若一张牌有多人进行和牌宣言，则按放铳者逆时针顺序优先进行，即头跳。
          2.   发出和牌宣言者若遇头跳未能真实和牌，不算入空行为。
          3.   若头跳者和牌宣言无效，视为诈和，仅罚取点棒，不重新对局，由下一位和牌宣言者和牌。
          4.   若和牌宣者系诈和，但遇头跳未真实和牌，只计空行为。
          5.   若所有和牌宣言者均为诈和，重新对局。
     4.   立直后可挑选和牌。和牌被打出时，若打出者下家已经开始摸牌而和牌者仍未发声，则视为见逃，进入振听状态。

2.   **和牌宣言**

     1.   和牌者在自摸、荣和时应进行发声，并展示手牌及和牌。
     2.   自摸和牌时，首先应展示摸到的牌。若将摸到的牌插入手牌后再进行和了宣言，亦可承认为和牌，但不计小符（听牌形相关的点数、跳符等），平和型和牌时不计平和役。
     3.   和牌宣言以发声视作开始，以动作结束视为终止，发声与动作可以同时开始，但禁止先行动作后发声。若仅有和牌宣言而无和牌行为，视为空行为；若仅有和牌行为而无和牌宣言，视为露牌。

3.   **和牌的确认**

     1.   当有对局者发出和牌宣言时，在所有对局者确认和牌前，不得破坏手牌、舍牌、牌山。
     2.   当有对局者宣告和牌或和牌点数时，由所有对局者一同判断和牌的有效性及点数的多少，随后其他对局者盖住自己的手牌，视为确认、承认和牌。

4.   **和程的复合**

     -   和程，即和了时的役种。

     1.   只有可以共存的役种方可复合计算。
     2.   不同的役满可以复合。

5.   **振听**

     1.   振听分为舍牌振听、立直振听与同巡振听。

          1.   当听牌时任一待牌（即使和了此牌无役）位于自己的牌河中时，是为舍牌振听。
          2.   当立直后见逃他家铳牌时，是为立直振听。
          3.   当任一待牌（即使和了此牌无役）为他家打出而未进行和牌时，是为同巡振听。同巡振听在自己再次摸牌时解除。

     2.   振听时仅可通过自摸和牌。

          

## 第六章 收支

1.   **原点**

     1.   竞技开始时，每家持有25000点，此为原点。
     2.   对局者有义务在竞技开始前确认自己的持点准确无误。

2.   **持点、顺位点**

     1.   持点超过30,000点以上的部分计为正分（+），不足的部分计为负分（-）。

     2.   持点以1000点=1pt的比率进行换算。

     3.   竞技顺位由半庄结束时合计得点的多少决定，持点从高到底排序。

     4.   顺位点指以半庄为单位，根据竞技顺位进行的分数加减。在本规定中，按以下方式表示：

          | 一位  | 二位  | 三位  | 四位  |
          | :---: | :---: | :---: | :---: |
          | +50pt | +10pt | -10pt | -30pt |

     5.   半庄以流局结束时，场上的供托归一位所有。

          1.   若南四局流局时存在同分并列一位，供托均分。
          2.   若供托点数不可整除，则离起家较近者取余数，其余部分均分（每根千点棒为400、300、300；每三千点时，点棒均分一次）。

     6.   半庄结束时若有多名玩家同点，则顺位点平分。

          1.   三人同点时，若顺位点无法整除，则离起家最近者取较大余数。

     7.   将持点与顺位点相加，得最终成绩（以pt结算）。

3.   **得点计算**

     1.   基于和牌的符数：

          | 底符 | 门清荣和符 | 边张、坎张或单骑 | 自摸符† |
          | :--: | :--------: | :--------------: | :-------: |
          | 20符 |     10     |       2符        |    2符    |

          †：除门清平和自摸外，所有自摸和牌全部加算2符的自摸符。

     2.   基于具体牌型的符数：

          | （符） | 中张牌 | 老头牌、客风牌 | 场风牌、自风牌、三元牌 |
          | :----: | :----: | :------------: | :--------------------: |
          |  雀头  |   0    |       0        |          2†          |
          |  明刻  |   2    |       4        |           4            |
          |  暗刻  |   4    |       8        |           8            |
          |  明杠  |   8    |       16       |           16           |
          |  暗杠  |   16   |       32       |           32           |

          †：连风牌（即一张牌同时作为自风与场风）只计2符。

     3.   最终符数†：

          1.   最终符数为底符、门清荣和符、牌形符、自摸符之合。
          2.   -   门清荣和：底符+门清荣和符+牌形符。
               -   门清自摸和：底符+自摸符+牌形符。
               -   荣和（副露）：底符+牌形符。
               -   自摸和（副露）：底符+自摸符+牌形符。

          †：七对子及龙七对子固定为25符。平和形副露荣和时，底符计为30符。

          3.   符数个位一律进十。

     4.   点数计算：

          1.   基本点为符数乘以二的番数加二之和的次方，即
               $$
               A=2^{h+2} f
               $$
               其中，A为基本点，h为番数，f为符数。

          2.   亲家荣和时，得到六倍基本点；亲家自摸时，各家支付二倍基本点。

          3.   子家荣和时，得到四倍基本点；子家自摸时，亲家支付二倍基本点，其余子家支付一倍基本点。

          4.   最终支付点数四舍五入保留两位有效数字。

          5.   当和牌番符为三番六十符及四番三十符时，按满贯计（切上满贯）。基本点取2000。

          6.   当三番牌符数大于六十符，四番牌符数大于三十符，及番数为五番时，为满贯。基本点取2000。

          7.   当番数为六番、七番时，为跳满。基本点取3000。

          8.   当番数为八番、九番、十番时，为倍满。基本点取4000。

          9.   当番数为十一番、十二番时，为三倍满。基本点取6000。

          10.   当番数为大于等于十三番时，为四倍满，即累计役满。基本点取8000。

          11.   役满基本点为8000。多倍役满复合时基本点按数目增加。

     7.   包牌者只需支付其包牌的点数。

          

## 第七章 处罚

### 犯规行为

-   空行为

    空行为指仅作发声，但未进行实际动作的行为，或发声与动作不一致。

-   露牌

    在非开立直的情况下，展示两张及以上手牌的行为；或在开立直情况下，展示与听牌无关的手牌的行为。

-   插入手牌

    摸打过程中，将摸到的牌插入手牌，而非置于手牌上方或手牌一侧，或置于手牌一侧时进行小手返（立直后属于出千）；若确实需要插入手牌，应如实报出手切摸切。

-   错摸

    开牌后，在摸牌时，摸取的牌非自己应当摸取的。

-   先摸

    在上家打牌之前即摸牌，或摸牌过快，以至在他家进行碰杠时，自己已提前摸牌。

-   撞山

    于码牌之后，于终局点棒交接完成前破坏牌山（展示两张及以上牌山之牌）的行为。

-   错行为

    错行为包括错吃、错碰与错杠。错吃指副露的三张牌无法组成顺子，错碰、错杠同理。

-   相公

    开牌后，在没有开杠的前提下，手牌在12张以下称小相公，在14张以上称大相公。

-   食替

    碰或吃时，打出的牌与副露的对子或搭子能组成面子的行为即为食替。

-   诈立直

    在未听牌的情况下立直，或流局宣布未听。

-   违规开杠

    立直后，在不满足开杠条件的情况下开杠。

-   诈和

    在未听牌的情况下宣布和牌，或在振听状态下荣和；若手牌未推开，视为空行为；若诈和遇他家头跳，不计诈和，计空行为。

-   出千

    在任意时刻故意调换，更改牌山、手牌及牌河。

### 处罚

1.   处罚包括罚取供托（罚立直棒）、罚取点棒（罚满贯）、放弃和牌、整局摸切、对局重开五种。
     1.   罚取供托者应支付1000点作为供托，由下一个和牌人获取。
     2.   罚取点棒者应当向其他三家支付点棒。向子家罚取时，应向其他子家支付2000点，向亲家支付4000点，合计8000点。向亲家罚取时，应向每位子家各支付4000点，合计12000点。
     3.   放弃和牌者在该局失去吃、碰、杠、立直、和牌的资格，并正常陪打至该局结束。
     4.   整局摸切者在该局失去任何改变手牌行为的资格，在该局内只得摸打。
     5.   对局重开指本局对局重新开始，除处罚外的点棒变动一律不计，本场数不改变。

2.   罚取供托及罚取点棒在犯规行为发生时处罚，若犯规行为系错行为，则在终局时处罚。

## 第八章 包牌

适用于包牌规则的情况下，若和牌者为自摸和，则包牌者支付全部点数；若和牌者为荣和，则包牌者与放铳者各支付一半点数。

1.   以下情况适用包牌规则：

-   包杠：打出的牌被其他竞技者用于开杠，或用岭上牌继续开杠，最终使对手岭上开花者。
-   大三元：打出第三种三元牌，使其他竞技者得以碰出（或杠出）第三次三元牌，最终使该竞技者大三元听牌者。

-   四喜和：打出第四种风牌，使其他竞技者得以碰出（或杠出）第四次风牌，最终使该竞技者四喜和（大四喜）听牌者。

-   四杠子：打出某牌，使其他竞技者得以进行第四次大明杠者。

2.   本场棒亦由包牌者承担。

3.   若和牌者的手牌复合多个役满，则包牌者只支付自己应承担的役满点数与本场棒，其余点数由他家正常支付。

## 简明规则一览

### 规定

1.   两万五千点起，**无终局点数要求**，**飞不了人**，以**尾亲下庄**结束。
2.   进行 吃、碰、杠、立直、荣和、自摸和 之竞技行为时**必须发声**。
3.   先发声，后进行动作，**不可缺少发声或缺少动作**。
4.   **无途中流局**，可以四杠，但是不能五；可以四风，可以四立，不准九种九。
5.   **严格暗杠规则**，立直后暗杠不得改变听牌、改变手牌分解方式、听牌分解方式、不得送杠。
6.   最多开杠四次，开杠立**即翻**宝牌。
7.   **最后一巡**也能**立**，除了摸海底的。
8.   有**包牌**，三元四喜四杠和==包杠==
9.   **没有多响**。
10.   **切上满贯**。
11.   **不要把摸到的牌插入手牌**！罚一千点！！
12.   最终得点比雀魂玩得大，一位更多，==四位更寄==。
13.   不要摆弄他家牌河，可以提醒别人放好。
14.   摔牌把牌把卓弄坏是要赔钱的，还容易吓着人。
15.   流程：
      1.   1、2、东、南、西、北六张牌定亲家及座次（每半庄做一次）。
      2.   洗牌，码牌，推牌
      3.   配牌
      4.   开牌（**避免在配牌时开牌**，开牌后出现相公需判罚，开牌前出现补牌即可）
      5.   摸牌、打牌、行牌
      6.   和牌、流局、听牌
      7.   结算


### 处罚

1.   犯规行为包括：

     **空行为**：发声后无行动，说吃做碰。
     **露牌**：露两张及以上手牌。
     **插入手牌**：摸到的牌要放在手牌两侧或上方，==除非自己报手切摸切==。
     错摸：拿错位置。
     **先摸**：别人没打你先摸，别人碰杠你还摸。
     **撞山**：露出两张及以上牌山牌。
     错行为：37吃2、6696暗杠。
     **相公**：牌数不对。
     **食替**：345吃2打5.
     **诈立**：未听立直、流局不报听（==流局可以不报听，只有立直不报听才有判罚，大吊车都可以不报听，但是-3000==）。
     **违规开杠**：开杠改变听牌、改变牌型、改变可能的听牌分解方式。
     **诈和**：未听和牌，振听荣和，==不推牌只算空行为==。
     出千：一次砍断一根手指。

### 役

1.   **有开立直、开两立直**，供托一千点。==需要展示也只能展示与听牌有关的牌，包含所有听牌分解方式==。

3.   **人和**，六番。

6.   番数上限为十三番，**无多倍役满**，但是不同役满可以复合。
